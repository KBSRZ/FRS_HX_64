C/S架构：
环境要求：window系统，，net4.0，mysql数据库

数据库建表文件：FaceAgainMysql_Set.sql
		FaceAgainMysql.sql
必须先执行FaceAgainMysql_Set.sql，后执行FaceAgainMysql.sql

运行：	配置海信网络科技.exe.config，将其中的数据库连接密码改为本地数据库密码
	运行海信网络科技.exe

注册人脸：进入设置界面，选择要录入的底库，点击登录，选择人脸注册参数，pitch，lown，yawn分别表示注册人脸的人脸旋转角度，仰头角度以及左右摇摆角度；选择合适的人脸质量阈值
	  进入注册页面，选择人脸文件夹，点击注册；0表示注册成功

1对1人脸比对：点击图片选择两张人脸图片进行相似度比对，点击比对显示结果
1对n人脸比对：进入设置界面选择要录入的底库，点击登录，点击左侧图片选择人脸图片进行相似度比对，右侧图片显示对应的最相似底库照片以及对应分数。

底库查看：进入设置界面，选择要查看的底库，点击登录，点击查看显示底库

监控：	进入设置见面，选择要录入的底库以及对应的摄像头，点击登录，选择人脸监控参数，包括三个人脸角度，人脸质量阈值以及告警阈值
	选择视频流选项，在右侧地址栏写入视频地址，点击开始按钮开始监控，抓拍的人脸会显示在下方人脸显示框，当监控到人脸相似度超过告警阈值时，会在右侧显示告警人脸以及相应的底库人脸
